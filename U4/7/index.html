Countable Nouns

Is there an apple? Yes, there is .

Are there any bananas? Yes, there are . How many are there? There is one . There are two / three / four / etc. We use: There are some / a few / a lot .

Uncountable Nouns

Is there any milk / cheese / yoghurt? Yes, there is.

How much is there? There is a lot . We use: There is some / a little / a lot .

Note:
